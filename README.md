# CareerNest - Job Search App

Hello! This is my Job Search application called **CareerNest**. I built this using **React** and **Vite** to help people find jobs in different companies. 

## Features
- **Login Page**: Secure login using JWT tokens.
- **Home Page**: A nice welcome screen with a description of the app.
- **Jobs Page**: (Coming soon/In progress) Where you can see all available job listings.
- **Protected Routes**: You can't see the home or jobs page without logging in first!
- **Logout**: Safely logout to clear your session.

## Technologies Used
- **React.js**: For building the user interface.
- **React Router**: For navigating between pages.
- **Bootstrap**: For making the app look good and responsive.
- **JS-Cookie**: To store the login token safely.

## How to Run the Project
1. Clone the repository to your computer.
2. Open the folder in VS Code.
3. Open the terminal and run:
   ```bash
   npm install
   ```
4. After installation, run:
   ```bash
   npm run dev
   ```
5. Click the link in the terminal (usually `http://localhost:5173`) to see the app!

## Note
I used the CCBP APIs for the backend part of this project.
